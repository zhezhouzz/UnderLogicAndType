Open Assert/Pose Normalization Analysis
=======================================

Goal
----

Track proof-local `assert` / `pose proof` facts in `Denotation` and
`ContextTyping` whose stated type still exposes normalizable open forms:

- `formula_open`
- `cty_open`
- `qual_open_atom`
- `lty_env_open_one (typed_lty_env_bind ...)`

The optimization rule for this pass is source normalization: when a local fact
is ordinary proof plumbing, state it in the normalized form before using it.
Raw opens should remain only in syntax/normalizer lemmas or in facts whose
logical subject is genuinely an opened type/body.

Scan command
------------

```sh
perl -0ne 'while(/(?:assert \([^\.]*?|pose proof \([^\.]*?)(?:formula_open|cty_open|qual_open|lty_env_open_one)[^\.]*\./sg){...}' \
  Denotation/theories/*.v ContextTyping/theories/*.v
```

  - `TypePersistArrow.v`, `TypePersistWandForward.v`,
    `TypePersistWandReverse.v`.

Batch B: result body assertions with `cty_open 0 y τr`

- Examples:
  - `TypeEquivArrowResultFirst.v`: `Hres_src_regular`,
    `Hres_tgt_regular`.
  - `TypeEquivTLet.v`: Arrow/Wand `Hres_*_regular`.
- These are partly semantic, because the result type is genuinely opened at
  the argument variable. Do not blindly remove them. Prefer normalizing
  surrounding env/term opens first; keep the opened type if it is the actual
  theorem subject.

Batch C: guard regular assertions for opened app result

- Examples:
  - `TypeEquivArrow.v`, `TypeEquivWand.v`;
  - `SoundnessApp.v`, `SoundnessAppD.v`, `SoundnessFix*.v`,
    `SoundnessLam*.v`.
- Many can likely be replaced by existing regular accessors from
  `ty_denote_gas 0` or `ty_guard_formula`; defer until result-first outer
  assertions are normalized.

Batch D: qualifier/open persistence and const facts

- Examples:
  - `TypePersistBase.v`, `ConstDenoteBase.v`,
    `SoundnessFixBase.v`.
- These often involve actual opened qualifiers, but some statements still
  expose `qual_open_atom 0 y ...` where a named normalized body abbreviation
  would be clearer. Defer to a Qualifier-specific cleanup.
