# Result-First Proof Optimization Plan

## Purpose

This optimization pass concentrates the repeated proof shape generated by
result-first denotation:

```coq
FForall (FImpl (expr_result_formula_at ... e result_slot) value_body)
```

The goal is to make result-first proofs read as:

1. open the result slot;
2. transport or build the result graph;
3. enter the value body;
4. apply the argument/result proof.

This pass does not delete lemmas.  Existing specialized lemmas should remain as
thin wrappers over the shared result-first helpers.

## Skill Guidance Used

- Prefer semantic helper statements over raw syntax helpers.
- Extract missing hard steps as named lemmas with short semantic statements.
- Do not replace semantic progress with larger shape-specific obligations.
- Use old proof routes for ideas, not as permanent dependencies.

## Concrete Optimization Work

1. Strengthen outer opening helpers.
   - `result_first_outer_graph_to_value_body`
   - `result_first_outer_ret_value_elim`
   - `result_first_outer_result_ext_elim`
   - `result_first_outer_projected_source`

2. Add value-body transport helpers that are not tied to a specific theorem.
   - `arrow_value_body_projection_transport`
   - `wand_value_body_projection_transport`
   - `persist_value_body_projection_transport`

3. Collect FV/scope facts for opened value bodies.
   - `formula_fv_open_arrow_value_body_obs`
   - `formula_fv_open_wand_value_body_obs`
   - analogous `CTPersist` value-body facts if needed

4. Replace duplicate manual proof skeletons.
   - In Denotation, replace long outer `FForall/FImpl` opening sequences.
   - In Soundness, keep typing-specific facts local, but call shared
     result-first opening helpers once the result extension/fresh slot is
     constructed.

5. Preserve theorem-specific wrappers.
   - Do not delete `lam_result_first_*`, `app_*`, `appd_*`, or `fix_*` lemmas.
   - Reprove them by calling the shared result-first helpers so downstream
     files do not churn.

## High-Value Candidate Replacements

  - Treat `CTPersist` as another result-first outer body.
