Unified Qualifier / Exact Atom Migration Tasks
============================================

Status: in progress on branch codex/concrete-primop-context.

Goal
----

Unify all Formula atoms around one store-level `qualifier`.  `FAtom q`
uses exact world denotation.  Guard-style atoms use the sugar
`FFiberAtom q := FFibVars (qual_vars q) (FAtom q)`.

Required durable lemma: first priority
--------------------------------------

Add a nontrivial semantic theorem for `FFiberAtom`:

  m |= FFiberAtom q <->
    every store in m satisfies q

The theorem must not be merely an unfold/sugar lemma.  It should expose a
usable store-level statement, hiding `FFibVars`, `formula_msubst_store`, and
`qualifier_exact_denote` internals.

This theorem must be implemented before continuing BasicDenotation migration.
All guard formulas (`basic_world_formula`, `context_ty_wf_formula`,
`expr_basic_typing_formula`, `expr_total_formula`, `expr_result_formula`)
should use this theorem instead of unfolding fiber/exact atom internals.
