Local Lemma Dependency Classification
=====================================

Scope
-----

This file classifies `Local Lemma` blocks in:

- `Denotation/theories/*.v`
- `ContextTyping/theories/Soundness*.v`

The scan is intentionally dependency-oriented rather than proof-length-oriented.
The key question is whether a local lemma really uses the current high-level
file's definitions, or whether it only uses lower-layer syntax/stdpp/resource
facts and should be moved down.
