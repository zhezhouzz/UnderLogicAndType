# Type Denote Recovery Status

## Current Shape

- `ty_denote_gas` now uses the original denotation environment `Σ`.
- `CTOver`, `CTUnder`, and `CTSum` use result graphs of the form:
  `expr_result_formula_at (dom Σ) ...`.
- `expr_result_formula_at` takes an `lvset`; the old atom-set wrapper
  `expr_result_formula_on X` remains only as `lvars_of_atoms X`.
- Fiber transport naming is back to `equiv`, not `refines`.
- `typed_fiber_equiv_on` observes `fv_cty τ`; the equality is exact on that
  fixed fiber, while result-graph fibering is controlled by `dom Σ` inside the
  type denotation.
